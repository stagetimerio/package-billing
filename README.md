# Stagetimer Billing

**Modern TypeScript package for Paddle billing integration** - This package is actively developed and replaces the legacy `@stagetimerio/planutils` package.

## Migration Status

This package provides:
- TypeScript types for Paddle payment platform entities
- Dynamic plan limit calculation from Paddle subscription/transaction items
- Dual CJS/ESM exports for modern JavaScript environments

The legacy `@stagetimerio/planutils` package is being deprecated. New code should use `@stagetimerio/billing` instead.

## Install

```sh
echo "@stagetimerio:registry=https://npm.pkg.github.com" >> .npmrc

npm i @stagetimerio/billing
```
